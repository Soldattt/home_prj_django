# Домашняя работа
Проект интернет-магазина разработан с помощью фреймворка Django,
## Установка
1. Клонируйте репозиторий:
```
git clone https://github.com/Soldattt/home_prj_django.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Начало работы
Для запуска программы, в терминале используйте команду запуска локального сервера 
```
python manage.py runserver 
```
## Работа программы
После открытия главной старницы вы можете перейти на страницу *Контакты* кликнув по соответствующей вкладке, где есть возможность отправить сообщение специалистам магазина, заполнив соответствующие поля для обратной связи
